# Chalo Golo

Chalo Golo is a goal-first learning app built with React and Vite. The name means "walk toward your goals": learning made simple, realistic, and personal.

## Tech Stack
- React
- Vite
- JavaScript
- HTML & CSS

## Installation

```bash
git clone https://github.com/Akz-Akz/chalo-golo.git
cd frontend
npm install
npm run dev
```
